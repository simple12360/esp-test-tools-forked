低功耗蓝牙阻塞测试
=================================

:link_to_translation:`en:[English]`

低功耗蓝牙阻塞测试评估无线设备在存在其他无线信号干扰的环境中的性能和稳定性，以确保其符合相关标准。

低功耗蓝牙阻塞测试方式
--------------------------------------

低功耗蓝牙阻塞测试可以在非信令（定频）模式下进行，选择一个固定频率来测试设备的抗干扰能力。测试步骤请参考 :doc:`./bt_ble_non_signaling_test`。

低功耗蓝牙阻塞测试也可以通过 DTM 模式进行，该模式允许对设备进行底层控制，便于引入干扰信号来评估设备的抗干扰能力。测试步骤请参考 :doc:`./ble_dtm_test`。
