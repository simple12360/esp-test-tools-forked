低功耗蓝牙阻塞测试
=================================

:link_to_translation:`en:[English]`

低功耗蓝牙阻塞测试评估设备在存在其他无线信号干扰的环境中的性能和稳定性，以确保其符合相关标准。

低功耗蓝牙阻塞测试方式
--------------------------------------

该测试可以通过以下两种模式进行：

1. **非信令模式**

在此模式下，选择一个固定频率来测试设备的抗干扰能力。测试步骤请参考 :doc:`./bt_ble_non_signaling_test`。

2. **DTM (Direct Test Mode) 模式**

DTM 模式允许对设备进行底层控制，从而引入干扰信号来评估其抗干扰能力。测试步骤请参考 :doc:`./ble_dtm_test`。
